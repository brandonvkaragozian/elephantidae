        Total: 275 cells
    
    Interpretation:
        - These 275 cells represent elephant movement range
        - Monitoring all = 100% coverage (unrealistic)
        - Baseline for evaluating camera placement
    
    Importance: Defines the universe of surveillance targets

METRIC 2: Total Conflict Zone Cells
    Definition: Cells where trajectories intersect human infrastructure
    Formula: COUNT(cells where visit_count > 0 AND 
                   (road_length_m > 0 OR rail_length_m > 0))
    Unit: Number of cells
    Range: 0 to 275
    
    Calculation:
        Type A - Trajectory + Road: 94 cells
        Type B - Trajectory + Railway: 16 cells  
        Type C - Trajectory + Settlement: 0 cells
        Total: 107 cells
        
    Interpretation:
        - These are highest-priority surveillance zones

        - 38.9% of all trajectory cells are conflict zones
        - Emphasis on these cells directly saves lives/property
    
    Importance: Defines critical zones requiring absolute coverage priority

    Definition: Cells in top 25% of visit frequency
    Formula: Cells with visit_count >= PERCENTILE(visit_count, 75)
    Unit: Number of cells

4.3 FAILURE MODE ANALYSIS

MIP Potential Failure Modes:
    1. Infeasibility: Budget too low, problem unsolvable
       Mitigation: Always feasible with budget ≥ max_coverage
    
    2. Numerical issues: Large coefficients cause rounding errors
       Mitigation: Normalize weights to [0,1]
    
    3. Timeout: Large problem doesn't solve in time
       Mitigation: Use warm-start with greedy solution

ML Potential Failure Modes:
    1. Poor feature selection: Wrong features lead to wrong importance
       Mitigation: Domain expert review features
    
    2. Data drift: Future elephant behavior different from training
       Mitigation: Retrain quarterly with new data
    
    3. Overfitting: Model memorizes training data patterns
       Mitigation: Cross-validation; use conservative hyperparameters
